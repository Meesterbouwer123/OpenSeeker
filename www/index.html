This server runs OpenSeeker

What is OpenSeeker?

OpenSeeker is a free and open source program and research project that scans for Minecraft servers on the internet. To do this, it connects to many internet-connected computers. This is harmless.

What does OpenSeeker do?

  • Connect to computers via TCP
  • Send small probe messages ("pings") to listening services
  • Temporarily join Minecraft servers in offline mode

What does OpenSeeker NOT do?

  • Scan for vulnerabilities
  • Perform stress tests or cause denial of service
  • Interact with web servers

I run a Minecraft server. How do I protect myself from bad actors?

  • Enable online mode, or use an auth plugin
  • Enable the whitelist
  • Run PaperMC
  • Ensure you are not vulnerable to BungeeCord spoofing
